# UI Mockups

## How to Review

Open `index.html` in a browser and use the navigation buttons to move between pages.

## Pages Included

- `index.html` - Entry point with links to mockups
- `dashboard.html` - Stage overview with status and actions
- `stage.html` - Stage detail with document list and preview
- `document.html` - Editor with split view preview
- `execution.html` - Streaming output console

## Navigation Flow

Dashboard → Stage Detail → Document Editor → Execution Console
